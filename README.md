# Nokiah_Space-Shooter

A retro-style arcade space shooter game, build using Python and Pygame, that you can play on your mobile device or desktop.

Features:

 - Start/Quit buttons
 - Theme selection (Dark, Neon, Retro)
 - Difficulty level selection (Easy, Medium, Hard)
 - Gameplay screen:
     - Player-controlled spaceship
     - Enemy ship with movement
     - Shooting system with bullets
     - Explosive effects
     - Current scores & high_scores
     - On-Screen buttons:
          - Move Left/Right
          - Fire
     - Game restart button


How to run:
 1. Install Python IDE
 2. Install Pygame
 3. Run the game:
      ⇒ nokiah_space_shooter.py
